# simkl-json-export
Script for exporting SIMKL data into a JSON format

Instructions:
* Go to https://simkl.com/settings/developer/ and create a new app with the return URI urn:ietf:wg:oauth:2.0:oob for pin auth
* Paste client id into field in conf.ini
* Run script and follow along
* Take the resulting watched.json and import to watcharr as if the file was a watcharr export
